# Why — epistemics

## Confidence tiers

| Tier | When | Phrasing |
|------|------|----------|
| Direct | Explicit text in a cited source | Present tense; quote or tight paraphrase + citation |
| Inferred | Combination of signals; not explicitly stated | “appears to”, “likely”, “suggests”; show the chain |
| Hypothesis | Multiple stories fit | List with evidence for/against; no forced winner |
| Unknown | Searched and empty, or unsearchable | Name the gap concretely |

## Rules

- No citation → not “What We Found.”
- Code shape alone is not intent (“checks null because it checks null”).
- Prefer contradictions surfaced over a smooth single story.
- Empty category searches are evidence (“no ticket discussed this threshold”).
